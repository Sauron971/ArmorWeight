# ArmorWeight Minecraft Plugin

This plugin adds a weight system for armor on a Minecraft server.

It allows server administrators to configure the weight of each armor piece and enable required plugin features through the configuration file.

---

## Features

* Adds weight mechanics for armor;
* Configurable weight for each armor piece;
* Flexible configuration system;
* Ability to enable or disable plugin functionality;
* Designed for Minecraft servers.

---

## Configuration

The weight of each armor part can be customized in the configuration file.

Example:

```yml
armor:
  helmet: 1
  chestplate: 3
  leggings: 2
  boots: 1
```

The plugin behavior can be adjusted according to server requirements.

---

## Installation

1. Download the plugin `.jar` file.
2. Place it into the server `plugins` folder.
3. Restart the Minecraft server.
4. Configure the plugin using the generated config file.

---

## Requirements

* Minecraft Server 1.20.6
* Bukkit / Spigot / Paper API

# ArmorWeight Minecraft Plugin

Данный плагин добавляет систему веса для брони на Minecraft сервер.

Плагин позволяет настроить вес каждой части брони и включить необходимый функционал через конфигурационный файл.

---

## Возможности

* Добавление механики веса для брони;
* Настройка веса каждой части брони;
* Гибкая система конфигурации;
* Возможность включать и отключать функционал плагина;
* Подходит для Minecraft серверов.

---

## Настройка

Вес каждой части брони можно изменить в конфигурационном файле.

Пример:

```yml
armor:
  helmet: 1
  chestplate: 3
  leggings: 2
  boots: 1
```

Настройки позволяют адаптировать работу плагина под требования сервера.

---

## Установка

1. Скачать файл плагина `.jar`.
2. Поместить его в папку `plugins` сервера.
3. Перезапустить Minecraft сервер.
4. Настроить плагин через созданный конфигурационный файл.

---

## Требования

* Minecraft сервер 1.20.6
* Bukkit / Spigot / Paper API.
